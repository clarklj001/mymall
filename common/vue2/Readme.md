Vue模块化开发
1.全局安装webpack
npm install webpack -g

2.全局安装vue脚手架
npm install -g @vue/cli-init

3.初始化vue项目
vue init webpack appname：vue脚手架使用webpack模板初始化一个appname项目

4.启动vue项目
npm start = npm run dev
